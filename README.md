# juliennecodes-final

## Requirements
* ruby version 2.7.2

## Instructions
* Clone project
* Go to project folder
* Set up database
  * go to server
  * type command - bundle install
  * type command - rails db:migrate
  * type command - rails db:seed
* Start Server
  * go to server
  * type command rails s
* Run react app
  * go to client
  * type command - npm install
  * type command - npm start
